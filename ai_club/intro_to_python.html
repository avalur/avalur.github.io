Why is Python called Python? The name comes from the British comedy group Monty Python, not the snake. And Monty Python is a British surreal comedy group that created the sketch comedy television show Monty Python's Flying Circus, which first aired on the BBC about 50 years ago. The creator of Python, Guido van Rossum, was a big fan of Monty Python's Flying Circus. And also he believed that a name that was short, unique, and slightly mysterious would be a good name for a programming language. Also, he thought that programming language should be not only convenient to use and easy to learn, but also fun to work with.
